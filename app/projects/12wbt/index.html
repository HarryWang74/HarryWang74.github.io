Background

12WBT is a subscription-based digital product designed to help people reach their fitness goals, such as losing weight. Their public site features various fitness programs and nutrition plans include online shop and blog. I was part of an agile team was involved to build their public site and responsible for create landing page, micro site and eDM for various promos.

Modularize front-end code

We modularized our SASS code to keep things organized and sharable. Creating page snippets with bootstrap+ liquid, so that content editors can use them very easier. E.g. { % include 'sign-up' % } will insert a member sign up form at any place across the site.

Mico site, Landing page and eDM

I responsible for create landing page, micro site and eDM for various promos and make sure form validation works properly, valid data been sent to backend.

Share/Exchange Data

12wbt member site build on Rails Application using mySQl, the public site build on Locomotive (a rails CMS) using MongoDB. Sometime we need to share/exchange data between two sites. Eg public site show case most popular recipes from member database. Sometime we need to extend locomotive engine to use models to access the database, other approach is to build API. It depends on what kind of data we need to handle and how to handle it.
