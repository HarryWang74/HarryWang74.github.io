Background

12WBT is a subscription-based digital product designed to help people reach their fitness goals, such as losing weight. 12WBT public site features various fitness programs and nutrition plans include online shop and blog. I was part of an agile team was involved to build 12WBT public site and responsible for create landing page, micro site and eDM for various campaigns.

Data Share/Exchange

12wbt member site build is Rails Application using mySQl, the public site build on Locomotive (a rails CMS) using MongoDB. Sometime we need to share/exchange data between two sites. Eg public site need to grab most popular recipes from member database. Sometimes we extended locomotive engine to access the database and use liquid (locomotive templating language) to render data in html. And sometimes we use JavaScript interacting with JSON APIs.

Mico site, Landing page and eDM

I responsible for create landing page, micro site and eDM for various campaigns and make sure form validation works properly, valid data been sent to backend.

Modularize front-end code

We modularized SASS code to keep things organized and sharable. Creating page snippets with bootstrap + liquid, so that content editors can use them very easier.
