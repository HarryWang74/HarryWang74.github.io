Front-end, CMS implementation

Rotacota site is not an Angular single page application but I used Angular Directive extensively to structure and modularize front-end code. For backend, we chosen Locomotive, an open source CMS for Rails, uses Liquid templating and MongoDB.

Generate PDF Catalogue

We extend the CMS so that client can generate a PDF product catalogue with barcode image. After download/print the catalogue, deals can easily to add production into barcode system by scan barcode image.
