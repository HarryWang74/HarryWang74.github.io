Three
ways to use SVG

Due to its small file size and scalable feature, SVG in web is getting more and more popular.

To embed SVG into web page as an image is the most common way. Chris Coyier published a nice article on css-tricks.com about this topic.

Another way to use SVG is icon fonts. Check out Massimo Cassandro’s tutorial on Sitepoint.com. Free online icon fonts generator such as icommon.io can meet most people’s need. But if you are looking for something more powerful, Glyphas is a more professional choice by font designers.

Pay extra attention if you need to add new icons and re-generate your icon fonts. Make sure old icon still use the same class name. Sometimes icons may get a few pixels offset after re-generation. Those issues will cause difficulty in adding new icons into current icon fonts.

The third way is to create interactive SVG with JS and CSS. See my example below. Treat path as a regular DOM element. But be aware of that jQuery add/remove class doesn’t work for path. You have to use .attr() instead and .setAttribute() in Vanilla JS.
