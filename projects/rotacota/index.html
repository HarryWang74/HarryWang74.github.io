Rotacota Site

RedAnt project / Design / Development

Front-end, CMS implementation

Rotacota site is not an Angular single page application but I used Angular Directive extensively to structure and modularize front-end code. For backend, we chose Locomotive, an open source CMS for Rails that used Liquid templating and MongoDB.

Generate PDF Catalogue

We extended the CMS so that clients can generate a PDF product catalogue with barcode images. After downloading the catalogue, dealers can easily add production into barcode system by scanning barcode image.
