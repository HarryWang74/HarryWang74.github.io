12WBT
Public Site

RedAnt Project / Development

Background

12WBT.com is a very popular health and fitness site, with a large community of highly engaged members. It actually includes two sites:

Public site, built on locomotive(a rails CMS) using MongoDB, showcases various fitness programs, nutrition plans, and fitness articles to general public.

Member site, is rails application using MySQL, provides members 12 weeks of exercises, food plans, shopping lists, live chat, forum and interactive tools.

I was a member of an agile team that built/maintain 12WBT public site.

Data Share/Exchange

We needed to perform data share/exchange between member site and public site. Two technical methods used to fulfill this task:

  • Extending locomotive engine to access MySQL
  • Using JavaScript interacting with JSON APIs

Micro site, Landing page and eDM

I was responsible for creating landing pages, micro sites and eDMs for various campaigns.

Modularize front-end code

We modularized SASS code to keep things organized and sharable.
