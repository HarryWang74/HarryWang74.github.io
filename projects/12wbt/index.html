12WBT
Public Site

RedAnt project / Development

Background

12WBT is a subscription-based digital product designed to help people reach their fitness goals, such as losing weight. 12WBT includes two sites:

Public site, built on locomotive(a rails CMS) using MongoDB, showcases various fitness programs and nutrition plans including online shop and blog to general public.

Member site, is rails application using MySQL, provides tailored fitness programs and nutrition advice for fee-paying users, and allow users to interact with other users on real time.

I was a member of an agile team that built 12WBT public site. My key responsibilities were to create landing pages, micro sites and eDMs for various campaigns and other maintenance tasks.

Data Share/Exchange

We needed to perform data share/exchange between member site and public site. Two technical methods used to fulfill this task:

  • Extending locomotive engine to access MySQL
  • Using JavaScript interacting with JSON APIs

Micro site, Landing page and eDM

I was responsible for creating landing pages, micro sites and eDMs for various campaigns and other maintenance tasks.
