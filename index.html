Recent project

Flu Reporter

In past a few years, I used a Google web service to check the flu trend in Sydney area in flu seasons. After Google stopped this service in September 2015, I realized such data would still be available on Google Trend. However, you have to set up complex search criteria every time when start search which is a bit annoying.

So I decided to build a small application that can automatically grab flu trend data from google API and share it with people who still want to use Google flu trend data. Read More
